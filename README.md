# DM-DE-Widerlegung der Grundmodelle  
## Eine strukturelle Neuinterpretation der Kosmologie

Dieses Repository dokumentiert eine vollständige, logisch konsistente Widerlegung der klassischen kosmologischen Modelle (Hubble, Friedmann, ΛCDM) und stellt eine neue Interpretation vor, die auf **lokaler Gravitation**, **Metrikvariation** und **diskreter Strukturphysik** basiert.  
Die Grundlage bilden zwei eigenständige wissenschaftliche Arbeiten:

- **TdR – Theorie der Realität**  
  DOI: https://doi.org/10.5281/zenodo.18727529  

- **Energie‑Revolution 8911 – Die Überwindung des Euler‑Widerstands**  
  DOI: https://doi.org/10.5281/zenodo.18778354  

Beide Arbeiten zeigen, dass Kontinuumsmodelle (Euler, Friedmann, Hubble) strukturelle Paradoxien erzeugen, wenn sie auf ein diskretes, inhomogenes Universum angewendet werden.  
Dieses Repository überträgt diese Erkenntnisse auf die Kosmologie.

---

## 🔍 Grundidee der neuen Interpretation

- **Gravitation statt Expansion**  
  Rotverschiebung misst Metrikvariation, nicht Geschwindigkeit oder Expansion.

- **Metrikvariation statt Dunkler Energie**  
  Voids → gestreckte Metrik → $c_{\text{eff}} > c$  
  Cluster → komprimierte Metrik → $c_{\text{eff}} < c$

- **Reale Dichteverteilungen statt Dunkler Materie**  
  Die „fehlende Masse“ entsteht durch falsche Mittelwertbildung, nicht durch unsichtbare Materie.

- **Diskrete Struktur statt Kontinuum**  
  Euler‑Modelle erzeugen künstliche Beschleunigung, Divergenzen und DM/DE‑Artefakte.

## 📡 3,3c als strukturelle Auflösung der Raumzeit

Die effektive Lichtgeschwindigkeit in Voids (~3,3c) ist kein neues Naturgesetz, sondern ein **struktureller Marker**:

- Sie zeigt die *niedrigste aktuell erkannte* Metrikstreckung.  
- Sie definiert die *momentane Auflösung*, mit der wir die Raumzeit beobachten können.  
- Sie macht Gravitation sichtbar, ohne neue Kräfte einzuführen.

**Analogie:**  
Mit lokalem c sehen wir das Universum wie ein 480p‑Bild.  
Mit 3,3c sehen wir es in HD.  
Wenn wir eines Tages eine Region mit 4,4c finden, steigt die Auflösung auf 2K.  
Bei 7c wären wir bei 4K.

Das Universum bleibt gleich – **unsere strukturelle Auflösung wird feiner**.
